Survival
========

A infinite exploration game.

Default Controls
----------------
Keyboard and mouse controls are hard-coded for now.

| Button     | Action            |
|------------|-------------------|
| W, A, S, D | Move horizontally |
| Space      | Move up           |
| Shift      | Move down         |
